◆ ite()

BDD_ID Manager::ite ( const BDD_ID  i,
const BDD_ID  t,
const BDD_ID  e 
)
overridevirtual

Method implements the if-then-else algorithm (recursive)

Parameters
i- decision node
t- output node if i is TRUE
e- output node if i is FALSE
Returns
id of the output node, if i is not a leaf node it iterates on itself until it finds a leaf node, after the roll back (by then it has a hierarchical structure) it checks the id of both legs of the i node for equal ids, if this happens the output is replaced with the id of one of the leg nodes, otherwise it checks the uniqueTable for equal nodes and replaces the output with the found node. If all these checks fail, it creates a new node with the collected data and returns the id of this node.

Implements ClassProject::ManagerInterface.
